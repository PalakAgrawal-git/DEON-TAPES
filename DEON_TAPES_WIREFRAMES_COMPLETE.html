INTERACTIVE ELEMENTS & SPECIFICATIONS

Call-to-Action Buttons

Primary CTA (Blue)
Used for: Main actions, forms, navigation to conversion pages
States: Default | Hover (darker blue) | Active/Clicked
Sizing: 40-50px height, 20-30px horizontal padding
Example: "Contact Us", "Explore Products", "Download Datasheet"
Secondary CTA (Light/Outline)
Used for: Alternative actions, secondary options
States: Default | Hover | Active
Example: "Learn More", "View Details", "Back to Products"
WhatsApp CTA (Green #25d366)
Used for: Direct messaging channel
Placement: Product detail pages, contact page, forms
Action: Opens pre-filled WhatsApp message with product/context info

Form Fields & Validation

Text Input
Height: 40px | Padding: 10px | Border: 1px solid #ddd
Focus state: Blue border (#2c5aa0), subtle shadow
Required fields: Marked with *
Textarea
Min-height: 120px | Resize: Vertical only
Same styling as text inputs
Dropdown/Select
Height: 40px | Appearance: Native select element
Options: Industry, Product Type, Country, etc.

Cards & Grids

Image: 150px height (aspect ratio 1:1)
